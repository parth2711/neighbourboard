# NeighbourBoard

A CLI-based local neighborhood discussion forum built using Java.

---

## Overview

NeighbourBoard is a command-line application that allows users to interact within a simulated local community. Users can create posts, comment, vote, and search threads.

---

## Features

* Create and view posts
* Upvote and downvote posts
* Add comments to posts
* Search posts by keyword
* Unique ID system for posts

---

## Tech Stack

* Java
* Object-Oriented Programming
* ArrayList

## How to Run

### 1. Clone the Repository

```
git clone https://github.com/parth2711/neighbourboard.git
cd neighbourboard
```

### 2. Compile

```
javac Main.java
```

### 3. Run

```
java Main
```

---

## How It Works

1. Enter username
2. Choose from menu
3. Perform actions:

   * Create post
   * View posts
   * Vote
   * Comment
   * Search

---

## Limitations

* No data persistence
* No multi-user support
* CLI-based interface
